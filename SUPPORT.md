# Support

## Community Support

Use GitHub Issues for reproducible bugs and feature requests.

Use GitHub Discussions for:

- setup questions
- workflow ideas
- roadmap feedback
- integration suggestions
- community examples

## Bug Reports

Please include:

- AIGate version
- Node.js version
- operating system
- exact command
- expected result
- actual result
- logs with secrets removed

## Security Reports

Do not open public issues for suspected vulnerabilities. Follow
[SECURITY.md](SECURITY.md).

## Commercial Support

Commercial support is planned for teams that need private onboarding, workflow
policy design, integrations, and enterprise governance. See
[docs/commercialization.md](docs/commercialization.md).
